# Color Flipper

A simple web application that changes the background color to a random color when the button is clicked.

## Usage

1. Open `index.html` in your web browser.
2. Click the "Flip Color" button to change the background color randomly.

## Files

- `index.html`: The main HTML file containing the structure and styles.
- `script.js`: The JavaScript file that handles the color flipping functionality.

## How it works

The JavaScript generates a random hexadecimal color code and applies it to the body's background color using CSS.

No dependencies required. Works in any modern web browser.
